# employee_leave_management_system
🧑‍💼 Employee Leave Management System

A web-based application developed to manage employee leave requests efficiently. This system provides separate dashboards for Admin (Owner) and Employees, ensuring smooth leave tracking and management.

🚀 Features
👨‍💻 Admin (Owner) Dashboard

View total leave requests

Approve / Reject leave applications

Manage departments

Add, update, and delete employees

Assign login credentials to employees

View reports of leave requests

👩‍💻 Employee Dashboard

Apply for leave with date selection

Automatic leave day calculation (using JavaScript)

View leave request status (Approved / Rejected / Pending)

Generate leave reports

Update profile details
